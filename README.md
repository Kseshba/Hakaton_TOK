Для того, чтобы воспользоваться разработанным сервисом, для начала необходимо установить среду разработки(PyCharm, Jupyter etc.) и все необходимые библиотеки: pytesseract, PyMuPDF(он же fitz), pillow(PIL), os, reportlab, PyPDF2, pdfplumber. Все эти библиотеки могут быть установлены через командную строку(pip install).
Необходимо поместить все файлы в одну папку, кроме того, в месторасположение библиотеки pytesseract необходимо добавить файл rus.traineddata(доступен по ссылке [1]), необходимый для работы с документами на русском языке. После этого нужно запусть программу app.py, и щелкнуть по выданному URL. На этот локальном сайте нужно нажать на кнопку "Выберите файл", выбрать тот файл, содержание которого Вам необходимо получить, после чего будет доступно обработать и скачать итоговый PDF-файл(кнопка "Добавить оглавление").
[1]https://github.com/tesseract-ocr/tessdata/blob/main/rus.traineddata
